== Description ==

This Plugin is for the [Shopping Cart Plugin](http://wordpress.org/extend/plugins/wp-e-commerce/) 
to manage the product stock. 
Currently it works only for the wp-e-commerce plugin version 3.7.8 or lower and products with limited quantity (physically on stock)

Features:

    * Two tables "Out of Stock" and "In Stock"
    * Tables searchable and limitable to your search phrase
    * For every product clickable link to product page

== Installation ==

To install the plugin to the following steps

1. Unzip the zip-file and upload the content to your Wordpress Plugin directory.
2. Activiate the plugin via the admin plugin page.
3. Go to Shopping Cart -> Inventory Manager
